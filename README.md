## Como começar

<br />

### CDN versão estável:

```bash
https://cdn.jsdelivr.net/gh/BeviProjects/BeviCSS@v1.0.6/dist/compressed/bevi.css
```

<br />

### CDN versão mais recente:

```bash
https://cdn.jsdelivr.net/gh/BeviProjects/BeviCSS/dist/compressed/bevi.css
```

<br />

### Link HTML

```html
<link
	rel="stylesheet"
	href="https://cdn.jsdelivr.net/gh/BeviProjects/BeviCSS@v1.0.6/dist/compressed/bevi.css"
/>
```

<br/>

### Link CSS

```css
@import url('https://cdn.jsdelivr.net/gh/BeviProjects/BeviCSS@v1.0.6/dist/compressed/bevi.css');
```
