## Como começar

<br />

### Link HTML

```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/BeviProjects/BeviCSS@v1.0.8/dist/compressed/bevi.css"
/>
```

<br/>

### Link CSS

```css
@import url('https://cdn.jsdelivr.net/gh/BeviProjects/BeviCSS@v1.0.8/dist/compressed/bevi.css');
```
